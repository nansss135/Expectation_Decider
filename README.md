Probability and Statistics Project using Python
Project Description

This project focuses on analyzing student exam performance using Probability and Statistics concepts with Python. The dataset contains information about students such as study hours, attendance, group discussion participation, previous test scores, and final exam results.

The project uses statistical methods, probability formulas, data visualization, and Python programming to understand factors affecting student performance.

Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn are used for data analysis and visualization.

Objectives
Understand basic probability concepts
Calculate empirical and conditional probabilities
Analyze random variables and probability distributions
Create contingency tables
Visualize data using graphs and charts
Study relationships between attendance, study habits, and exam performance

Results and Conclusion

The project analysis shows that:

Higher attendance increases chances of passing.
Students participating in group discussions perform better.
Study hours positively affect exam scores.
Previous test scores influence final performance.

The project demonstrates how Probability and Statistics can be applied to real-world educational data analysis.
